cli

Function - cli.addCommand

Description: The addCommand function registers a new command to the command pool.

Returns: A Command object to set additional information, eg. flags.

Name Type Description
path string The command name. For nested commands, separate each name with a dot.
callback function The function that will be called when the command is used.

Function - cli.log

Description: An alternative to console.log() with color styling capabilities.

Name Type Description
args string[] An array of strings to be written to stdout.

Function - cli.promptInput

Description: Prompts for input to stdout and stdin.

Returns: A promise with the input string.

Name Type Description
message string The message that comes before the input.
censored boolean Whether the input should be censored or not. Useful for passwords.

Function - cli.promptMenu

Description: Prompts a list of options to stdout that is interactable.

Returns: A promise with the selected option. In case an array of strings were provided as items, it will return the index of the selected item. Otherwise it will return the value of the selected item.

Name Type Description
items MenuParams An array of strings or an array of { text: string, value: any }

Method - Command.addFlag

Description: Registers a flag to a command.

Returns: A command object for chaining.

Name Type Description
name string The name of the flag.
type FlagTypeNames The type of value this flag will accept. Any by default.
required boolean Is this flag required or optional?
